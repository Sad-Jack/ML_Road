# 52. K-Means

## Зачем это нужно

K-Means — самый популярный алгоритм кластеризации. Он прост, быстр и даёт интерпретируемые результаты. Используется для сегментации клиентов, сжатия изображений, создания признаков и анализа данных. Понимание K-Means открывает двери в мир unsupervised learning.

## Простая интуиция

Представьте, что вы расставляете 3 флага на карте городов. Каждый город «идёт» к ближайшему флагу. Потом флаги перемещаются в центры своих групп. И так по кругу, пока флаги не перестанут двигаться. Флаги — это центроиды, города — точки данных.

## Что это такое

- **K-Means** — алгоритм, разбивающий данные на K кластеров, минимизируя расстояния внутри кластеров
- **K** — число кластеров, задаётся вручную
- **Центроид** — среднее значение всех точек в кластере
- **Инерция (inertia)** — сумма квадратов расстояний от точек до их центроидов (чем меньше, тем лучше)
- **Метод локтя (Elbow method)** — подбор оптимального K по графику инерции

## Как это работает

```python
from sklearn.cluster import KMeans

kmeans = KMeans(n_clusters=4, random_state=42, n_init=10)
kmeans.fit(X)

labels = kmeans.predict(X)
print("Центроиды:", kmeans.cluster_centers_.round(2))
print("Инерция:", round(kmeans.inertia_, 1))

# Метод локтя: обучаем K-Means для K=1..10, строим inertia[K]
# Ищем "перегиб" — точку, где снижение инерции резко замедляется
# inertia[1] >> inertia[4] ≈ inertia[5]  →  оптимальное K ≈ 4
```

## Простой пример

Сеть розничных магазинов хочет сегментировать 3 000 клиентов по двум признакам: «число визитов в месяц» и «средний чек». Запускают K-Means с K=2, 3, 4, 5 и строят метод локтя. При K=4 инерция резко перестаёт падать — оптимум найден. Четыре центроида описывают сегменты: «редко/дёшево», «редко/дорого», «часто/дёшево», «часто/дорого». Это actionable сегментация для маркетинга.

## Краткий итог

- K-Means чувствителен к инициализации и предполагает сферические кластеры: всегда проверяй разные k и смотри на визуализацию
- K нужно задавать вручную, метод локтя помогает выбрать оптимальное значение
- Алгоритм предполагает сферические кластеры одинакового размера
- Нужна нормализация данных: K-Means чувствителен к масштабу признаков

## Ключевые термины урока

- **K-Means** — алгоритм кластеризации, итеративно назначающий точки к ближайшему центроиду.
- **Центроид** — центр кластера; среднее по всем точкам, входящим в него.
- **Инерция (inertia)** — сумма квадратов расстояний до центроидов; мера качества кластеризации.
- **Метод локтя (Elbow method)** — поиск оптимального K по точке «перегиба» на графике инерции.
