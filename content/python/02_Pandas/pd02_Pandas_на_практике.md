# 02. Pandas на практике

## Зачем это нужно

Pandas — основной инструмент работы с данными в Python. Прежде чем обучать модель, нужно уметь смотреть на данные: выбирать нужные колонки, фильтровать строки, искать странные значения.

## Простая интуиция

Pandas — это как Excel, но в коде. Три главных действия: выбрать колонки (как SELECT в SQL), отфильтровать строки (как WHERE), посмотреть на структуру (сколько строк, какие типы).

## Что это такое

Pandas — библиотека Python для работы с табличными данными. В первом уроке Python-трека мы создали DataFrame — теперь научимся с ним работать.

## Как это работает

```python
import pandas as pd

df = pd.DataFrame({
    "age": [25, 31, 28, 40],
    "city": ["Berlin", "Hamburg", "Berlin", "Munich"],
    "bought": [1, 0, 1, 0]
})

# Смотрим на структуру
df.head()        # первые 5 строк
df.columns       # названия колонок
df.shape         # (4, 3) — 4 строки, 3 колонки

# Выбираем колонки
df["age"]                  # одна колонка (Series)
df[["age", "city"]]        # несколько колонок (DataFrame)

# Фильтруем строки
df[df["age"] > 30]
df[df["city"] == "Berlin"]
df[(df["city"] == "Berlin") & (df["age"] > 26)]  # AND — используй &, не and
```

Важно: при нескольких условиях каждое берём в скобки, `&` вместо `and`, `|` вместо `or`.

## Простой пример

Выделить X и y:

```python
X = df[["age", "city"]]
y = df["bought"]
```

Это аналог SQL: `SELECT age, city FROM df WHERE ... → X`; `SELECT bought → y`.

## Краткий итог

- `df.head()`, `df.shape`, `df.columns` — первый взгляд на данные.
- Одна колонка: `df["col"]`, несколько: `df[["col1", "col2"]]`.
- Фильтрация: `df[df["col"] > value]`, несколько условий через `&` и `|`.
- Сначала изучи данные — только потом думай о модели.

## Ключевые термины урока

- **Индексирование** — доступ к данным через `df["col"]` (Series) или `df[["col1", "col2"]]` (DataFrame).
- **Фильтрация** — отбор строк по условию: `df[df["col"] > value]`.
- **df["col"]** — возвращает колонку как Series.
- **df[["col"]]** — возвращает колонку как DataFrame (двойные скобки).
- **Булева маска** — массив True/False для фильтрации строк по условию.
