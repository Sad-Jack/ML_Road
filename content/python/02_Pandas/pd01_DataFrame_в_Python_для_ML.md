# 01. DataFrame в Python для ML

## Зачем это нужно

Почти любая ML-задача начинается с данных в табличном виде: CSV-файл, выгрузка из SQL, Excel. В Python для работы с таблицами используют `pandas` и его основную структуру — DataFrame.

## Простая интуиция

DataFrame — это таблица, как в Excel. Каждая строка — один объект (один клиент, одна квартира, одна транзакция). Каждая колонка — одно свойство объекта (возраст, площадь, цена). Это главное правило: строка = объект, колонка = свойство.

## Что это такое

**DataFrame** — это таблица данных в Python (библиотека pandas). Строки — объекты, колонки — их атрибуты. Из DataFrame мы выделяем X (признаки) и y (target) для обучения модели.

## Как это работает

Создать DataFrame и посмотреть на него:

```python
import pandas as pd

df = pd.DataFrame({
    "experience_years": [1, 2, 3, 4],
    "english_level": [2, 3, 4, 4],
    "salary": [70000, 90000, 120000, 150000]
})

print(df.head())   # первые строки
print(df.shape)    # (4, 3) — 4 строки, 3 колонки
```

Выделить X и y:

```python
X = df[["experience_years", "english_level"]]
y = df["salary"]
```

Важно: одна колонка — `df["salary"]`, несколько колонок — `df[["col1", "col2"]]`.

## Простой пример

| experience_years | english_level | salary |
|-----------------|---------------|--------|
| 1 | 2 | 70000 |
| 2 | 3 | 90000 |
| 3 | 4 | 120000 |

Три строки = три объекта (кандидата). Три колонки = три свойства. Первые две — признаки (X), третья — target (y).

## Практика

1. В DataFrame что обычно означает одна строка, а что означает одна колонка?
2. Почему перед обучением модели важно сначала посмотреть на данные через `df.head()`?
3. Есть таблица с колонками `age`, `city`, `orders_count`, `bought`. Если `bought` — это target, какие колонки попадут в X?
4. Чем отличается `df["salary"]` от `df[["salary"]]`?
5. Ты открыл CSV-файл и видишь 10 000 строк и 12 колонок. Что ты проверишь первым перед обучением модели?

## Краткий итог

- Умение читать и преобразовывать DataFrame — основа всего: без этого нельзя дойти до обучения модели.
- Строка = один объект, колонка = одно свойство.
- Три команды дают первый срез данных: сколько строк, какие колонки, как выглядит содержимое — этого достаточно чтобы начать.
- Из DataFrame выделяем X и y: `X = df[["col1", "col2"]]`, `y = df["target"]`.
- Сначала смотри на данные — потом обучай модель.

## Ключевые термины урока

- **DataFrame** — двумерная таблица данных в pandas; основная структура для работы с ML-данными.
- **Series** — одномерный массив в pandas; отдельная колонка DataFrame.
- **df.head()** — вывод первых 5 строк для быстрого первого взгляда на данные.
- **df.shape** — кортеж (число строк, число колонок) DataFrame.
- **df.columns** — список названий колонок DataFrame.
