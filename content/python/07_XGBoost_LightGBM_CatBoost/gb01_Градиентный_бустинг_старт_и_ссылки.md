# 01. XGBoost, LightGBM, CatBoost: старт и полезные ссылки

## Зачем это нужно

Градиентный бустинг по деревьям — один из самых сильных базовых инструментов для табличных данных. Во многих задачах именно эти библиотеки дают лучший practical baseline.

## Что важно понять в этом модуле

- чем бустинг отличается от одного дерева решений;
- когда выбирать `XGBoost`, `LightGBM` или `CatBoost`;
- как работать с категориальными признаками, переобучением и early stopping;
- какие гиперпараметры влияют на качество сильнее всего.

## Как проходить

Не изучай три библиотеки как три разных мира. Сначала пойми общую идею бустинга и один базовый пайплайн, а потом уже смотри различия в API и сильные стороны каждой реализации.

## Краткий итог

Для табличного ML это обязательный стек. Если ты не умеешь быстро поднимать baseline на бустинге, ты теряешь один из самых практичных инструментов в индустрии.
