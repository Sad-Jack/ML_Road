# 84. Что такое Deep Learning

## Зачем это нужно

Deep Learning — это то, что стоит за современными AI-прорывами: ChatGPT, Midjourney, AlphaFold, Tesla Autopilot. Понимание базовых принципов DL позволяет ориентироваться в этом мире и понимать, что возможно, а что нет.

## Простая интуиция

Обычная нейросеть — это один слой обработки. Deep Learning — это много слоёв, каждый из которых строит более абстрактные понятия. Для изображения: пиксели → края → формы → детали (нос, глаза) → лица. Каждый слой «понимает» чуть больше, чем предыдущий.

## Что это такое

- **Deep Learning** — нейронные сети с большим числом слоёв (от 10 до сотен)
- **CNN (Convolutional Neural Networks)** — для изображений и видео; учатся видеть локальные паттерны
- **RNN (Recurrent Neural Networks)** — для последовательностей (текст, временные ряды); есть «память»
- **Transformer** — архитектура на основе attention, заменившая RNN для большинства NLP задач
- **Transfer Learning** — использование предобученных моделей (ImageNet, GPT) для новых задач

## Как это работает

```python
import torch.nn as nn

# Deep = несколько слоёв, каждый строит более абстрактные признаки
model = nn.Sequential(
    nn.Linear(20, 64),   # входной слой → скрытый
    nn.ReLU(),
    nn.Dropout(0.3),     # регуляризация: случайно отключаем нейроны при обучении
    nn.Linear(64, 32),   # скрытый → скрытый
    nn.ReLU(),
    nn.Linear(32, 1),    # скрытый → выход
    nn.Sigmoid()         # вероятность (0..1) для бинарной классификации
)

total_params = sum(p.numel() for p in model.parameters())
print(f"Параметров: {total_params:,}")  # всё это учится из данных
```

## Простой пример

Исследователи создают классификатор рентгеновских снимков лёгких (пневмония/норма). Датасет: 5 200 снимков. С нуля (обучение CNN): accuracy 72% после 50 эпох, обучение 3 часа на GPU. Transfer Learning (ResNet-50, предобученный на ImageNet): accuracy 94% после 10 эпох дообучения только последних слоёв, обучение 20 минут. ResNet-50 уже «знает» края, текстуры, формы из 1.2 миллиона изображений — осталось только научить его отличать признаки пневмонии.

## Краткий итог

- Deep Learning = многослойные нейросети с автоматическим извлечением признаков
- Выбор архитектуры определяется типом данных: для нового проекта не начинай с нуля — Transfer Learning сэкономит месяцы
- Transfer Learning позволяет использовать мощные модели даже при малых данных
- DL требует много данных и GPU, но при этих условиях бьёт классический ML

## Ключевые термины урока

- **Deep Learning** — многослойные нейросети, автоматически извлекающие признаки из сырых данных.
- **CNN (Convolutional Neural Networks)** — архитектура для изображений через локальные фильтры.
- **RNN (Recurrent Neural Networks)** — архитектура для последовательностей через скрытое состояние.
- **Transformer** — универсальная архитектура на основе attention; доминирует в современном DL.
- **Transfer Learning** — использование предобученной модели как основы для новой задачи.
