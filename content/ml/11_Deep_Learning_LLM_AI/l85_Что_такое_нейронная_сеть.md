# 85. Что такое нейронная сеть

## Зачем это нужно

Нейронные сети лежат в основе современного AI: распознавание речи, перевод, генерация изображений, ChatGPT. Понять базовую идею нейросети — значит получить ключ к пониманию всего этого. Это не магия, а математика: числа, умножения, активации.

## Простая интуиция

Представьте детектор спама. Одно правило — «если есть слово БЕСПЛАТНО — спам» — часто ошибается. А если взять 100 слабых правил и объединить их с правильными весами — получается надёжный детектор. Нейронная сеть — это тысячи таких «правил» в слоях, каждое из которых обнаруживает свой паттерн.

## Что это такое

- **Нейрон** — вычислительная единица: принимает числа, умножает на веса, суммирует, применяет нелинейную функцию
- **Веса (weights)** — числа, которые сеть учит в процессе обучения
- **Слой (layer)** — группа нейронов, обрабатывающих одни и те же входные данные
- **Архитектура**: Input layer → Hidden layers → Output layer
- **Активационная функция** — нелинейность после суммирования (ReLU, sigmoid, tanh)

## Как это работает

Нейрон получает входные числа, умножает каждое на свой вес, суммирует результаты, добавляет константу (bias) и применяет функцию активации. Несколько нейронов, обрабатывающих одни и те же данные, образуют слой.

Структура нейросети:

```
Входы → [Input layer] → [Hidden layer 1] → [Hidden layer 2] → [Output layer] → Предсказание
```

Каждый нейрон в скрытом слое «смотрит» на все выходы предыдущего слоя и формирует своё представление. Нейроны одного слоя не связаны между собой — только с соседними слоями. Именно глубина (несколько скрытых слоёв) позволяет сети улавливать сложные зависимости в данных.

## Простой пример

Простая нейросеть для предсказания стоимости квартиры: входной слой с 5 нейронами (площадь, этаж, район, год постройки, число комнат), один скрытый слой с 16 нейронами (ReLU), выходной нейрон (цена). Всего: 5×16 + 16 + 16×1 + 1 = 113 обучаемых параметров. На датасете из 2 000 квартир сеть обучается за 200 эпох. Каждый нейрон в скрытом слое «специализируется»: один реагирует на сочетание «большая площадь + центральный район», другой — на «новый дом + высокий этаж».

## Практика

1. Из чего состоит один нейрон? Какие вычисления он выполняет и какова роль функции активации?
2. Почему нейросеть из одного нейрона принципиально слабее, чем сеть с несколькими слоями, даже если у одного нейрона очень много входов? В чём роль нелинейной активации?
3. Вы строите нейросеть для классификации изображений котов и собак. Сколько нейронов должно быть в выходном слое и какая функция активации подходит для последнего слоя?
4. В уроке «Гиперпараметры модели» мы разделяли параметры, которые модель учит из данных, и гиперпараметры, которые задаются вручную. Как это различие проявляется в нейросети: что является параметрами, а что — гиперпараметрами?
5. В уроке «Overfitting и Underfitting» переобучение объяснялось через сложность модели. Как количество нейронов и слоёв в нейросети влияет на риск переобучения и на каком этапе это нужно контролировать?

## Краткий итог

- Нейрон = взвешенная сумма входов + нелинейная активация
- Несколько слоёв нейронов = нейронная сеть
- Нелинейные активации (ReLU) позволяют выучить сложные зависимости
- Обучение сети — оптимизация весов на данных; как именно это происходит — тема следующего урока

## Ключевые термины урока

- **Нейрон** — вычислительная единица: `output = activation(w·x + b)`. Без нелинейной активации стек из тысячи нейронов эквивалентен одному линейному слою.
- **Веса (weights)** — единственное, что «знает» сеть; всё обучение сводится к нахождению нужных чисел для миллионов весов. Инициализируются случайно и подбираются в процессе обучения.
- **Слой (layer)** — параллельная обработка: нейроны одного слоя не связаны между собой, только с предыдущим и следующим слоем. Именно эта структура делает сеть модульной и позволяет обучать каждый слой независимо.
- **Активационная функция** — ключ к нелинейности; без неё нейросеть — просто матричное умножение. ReLU(x) = max(0, x) — стандарт для скрытых слоёв; sigmoid — только для выходного при бинарной классификации.
