# 92. Что такое AI Agent

## Зачем это нужно

LLM, который только отвечает на вопросы, — это чат-бот. AI Agent — это LLM, который может планировать, использовать инструменты и выполнять многошаговые задачи автономно. Это следующий уровень применения LLM: от ответов к действиям.

## Простая интуиция

Обычный LLM — как консультант, который отвечает на вопросы. AI Agent — как ассистент, которому можно сказать «организуй мне командировку» и он сам найдёт рейсы, забронирует отель, отправит письма и внесёт встречи в календарь. Разница — в способности действовать, а не только советовать.

## Что это такое

- **AI Agent** — система: LLM + инструменты + память + планирование
- **Инструменты (tools)** — функции, которые LLM может вызывать (поиск, API, код, база данных)
- **ReAct (Reason + Act)** — паттерн: Рассуждение → Действие → Наблюдение → Рассуждение...
- **Память**: контекстная (в промпте), долгосрочная (vector store), процедурная (память о навыках)
- **Планирование** — разбивка сложной задачи на шаги
- **Human-in-the-loop** — контроль человека на критических шагах

## Краткий итог

- AI Agent = LLM + инструменты + память + планирование
- ReAct паттерн: Рассуждение → Действие → Наблюдение — базовый цикл агента
- Инструменты расширяют возможности: поиск, код, API, базы данных
- Human-in-the-loop обязателен для критических и необратимых действий

## Ключевые термины урока

- **AI Agent** — система на основе LLM, способная самостоятельно планировать и использовать инструменты.
- **Инструменты (tools)** — внешние функции, которые агент может вызывать (поиск, код, API).
- **ReAct (Reason + Act)** — паттерн: рассуждение → действие → наблюдение результата → повторение.
- **Память** — механизм сохранения контекста между шагами или сессиями агента.
- **Human-in-the-loop** — включение человека в процесс для проверки критических решений.
