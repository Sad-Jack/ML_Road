# 83. Как нейросеть учится на ошибках

## Зачем это нужно

Ты уже знаешь, что нейросеть — это слои с весами (урок 82). Но как она вообще учится? Откуда берутся правильные веса? Ответ — в механизме обучения: loss, градиентный спуск и backpropagation. Без этого понимания нейросети остаются чёрным ящиком.

## Простая интуиция

Представь: ты бросаешь мяч в ворота и промахиваешься. Ты смотришь на ошибку (насколько промахнулся) и чуть корректируешь бросок. Повторяешь. Постепенно попадаешь точнее.

Нейросеть делает то же самое:
1. Делает предсказание
2. Считает, насколько ошиблась (loss)
3. Понимает, как надо изменить веса, чтобы ошибка уменьшилась (градиент)
4. Делает маленький шаг в правильном направлении
5. Повторяет миллионы раз

## Что это такое

**Loss (функция потерь)** — число, которое показывает, насколько плохо нейросеть справилась. Чем больше — тем хуже. Цель обучения: минимизировать loss.

Примеры:
- MSE (Mean Squared Error) — для регрессии
- Cross-Entropy — для классификации

**Веса (weights)** — числа в каждом нейроне, которые нейросеть подстраивает в процессе обучения. Именно в них хранится всё, что модель «знает».

**Градиент** — направление, в котором loss растёт быстрее всего. Чтобы его уменьшить, надо двигаться в **противоположную** сторону.

**Gradient Descent (градиентный спуск)** — алгоритм: обновляй веса маленькими шагами против градиента.

```
новый_вес = старый_вес − learning_rate × градиент
```

**Learning rate** — размер шага. Слишком большой — перепрыгнешь минимум. Слишком маленький — будешь учиться вечно.

**Backpropagation** — алгоритм для вычисления градиентов через всю сеть. По правилу цепочки (chain rule) он считает, как каждый вес во всех слоях повлиял на итоговую ошибку.

## Как это работает

```
Для каждого батча данных:
  1. Forward pass:  X → слои → предсказание ŷ
  2. Считаем loss:  L = loss(ŷ, y_true)
  3. Backward pass: backprop считает градиенты ∂L/∂w для всех w
  4. Update:        w = w − lr × ∂L/∂w
Повторить для всех батчей (= 1 эпоха)
Повторить много эпох
```

Это и есть SGD (Stochastic Gradient Descent) — основа обучения всех нейросетей.

**Батч и эпоха:**
- **Батч** — небольшая подвыборка данных (например, 32 примера). Считаем градиент на батче, а не на всём датасете — быстрее.
- **Эпоха** — один полный проход через все обучающие данные.
- Обычно обучают 10–100+ эпох.

## Простой пример

Нейросеть учится распознавать рукописные цифры на 60 000 изображениях. Размер батча — 64. За одну эпоху происходит 60 000 / 64 ≈ 937 обновлений весов. На первой эпохе loss = 2.31 (случайные предсказания), accuracy 11%. К 10-й эпохе loss = 0.08, accuracy 97.4%. Learning rate = 0.001 — маленький, но стабильный. Попытка с lr = 0.1: loss скачет и не сходится. Попытка с lr = 0.00001: сходится, но нужно 200 эпох вместо 10.

## Краткий итог

Нейросеть учится итеративно: один цикл forward–backward–update не меняет веса заметно, но тысячи шагов превращают случайную инициализацию в работающую модель. Learning rate — самый критичный гиперпараметр: слишком большой — расходимость, слишком маленький — обучение встанет.

## Ключевые термины урока

- **Loss (функция потерь)** — мера ошибки модели; то, что минимизируется при обучении.
- **Градиент** — вектор, показывающий направление роста loss; обновление идёт в обратную сторону.
- **Gradient Descent** — алгоритм обновления весов в направлении, уменьшающем loss.
- **Learning rate** — размер шага при обновлении весов.
- **Backpropagation** — алгоритм вычисления градиентов от выхода к входу нейросети.
- **Эпоха** — один полный проход по всем обучающим данным.
- **Батч** — подмножество данных для одного шага обновления весов.
