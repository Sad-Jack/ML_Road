# 81. AI, ML, Deep Learning и Generative AI: кто внутри кого

## Зачем это нужно

Слова AI, ML, Deep Learning, нейросети, LLM, GenAI — звучат повсюду, и часто их используют как синонимы. Но каждое из них означает конкретную вещь. Понять иерархию — значит не путаться в разговорах, статьях и вакансиях.

## Простая интуиция

Представь матрёшку:

```
AI (искусственный интеллект)
  └── ML (машинное обучение)
        └── Neural Networks (нейронные сети)
              └── Deep Learning (глубокое обучение)
                    └── Generative AI (генеративный AI)
                          └── LLM (большие языковые модели)
```

Каждый следующий уровень — это **подмножество** предыдущего. Не замена, а специализация.

## Что это такое

**AI (Artificial Intelligence)** — широкая область: любая система, которая имитирует человеческий интеллект. Включает логику, экспертные системы, поиск, планирование — и ML тоже.

**ML (Machine Learning)** — подраздел AI, где система **учится на данных**, а не запрограммирована жёсткими правилами. Линейная регрессия, деревья, бустинг — всё это ML.

**Нейронные сети (Neural Networks)** — подраздел ML, вдохновлённый биологическими нейронами. Состоят из слоёв, которые извлекают признаки автоматически.

**Deep Learning** — нейронные сети с **много слоями** (глубокие). Работают особенно хорошо с изображениями, аудио, текстом. Требуют много данных и вычислений.

**Generative AI** — подраздел Deep Learning, где модель **создаёт новый контент**: текст, изображения, аудио, код. Не просто классифицирует, а генерирует.

**LLM (Large Language Model)** — конкретный тип генеративного AI для работы с текстом. GPT-4, Claude, Gemini — примеры LLM.

## Как это работает

```
AI
├── Экспертные системы, логика, поиск (не ML)
└── ML
    ├── Линейные модели, деревья, бустинг (не нейросети)
    └── Neural Networks
        ├── Shallow Networks (1–2 слоя)
        └── Deep Learning (много слоёв)
            ├── CNN — изображения
            ├── RNN / LSTM — последовательности
            └── Transformer — универсальная архитектура
                └── Generative AI
                    ├── LLM — текст (GPT, Claude)
                    ├── Diffusion — изображения (DALL-E, Stable Diffusion)
                    └── Multimodal — текст + изображения
```

## Простой пример

Компания объявляет: «Мы используем AI для анализа отзывов». Что это значит конкретно? Уточнение: «Логистическая регрессия на TF-IDF признаках» — это ML, но не Deep Learning. «Дообученный BERT» — это Deep Learning. «GPT-4 через API» — это LLM, то есть Generative AI → Deep Learning → ML → AI. Все четыре утверждения технически верны, но несут разный смысл: объём данных, нужные вычисления, интерпретируемость, стоимость. Понимание иерархии позволяет читать между строк.

## Практика

1. Нарисуй иерархию AI → ML → Deep Learning → Generative AI → LLM своими словами, без подсказок. Что ты помнишь на каждом уровне?

2. Классификатор спама на логистической регрессии — это AI? ML? Deep Learning? Объясни почему каждый ответ верен или нет.

3. Почему Deep Learning — это подмножество ML, а не отдельная область? В чём принципиальное сходство?

4. Чем Generative AI отличается от обычного Deep Learning для классификации? Приведи пример обоих.

5. GPT-4 — это LLM. Назови три уровня иерархии, к которым он одновременно относится.

## Краткий итог

Иерархия AI → ML → Deep Learning → Generative AI → LLM помогает понять: любой LLM — это нейросеть, обученная на данных, со всеми присущими ограничениями. Проблемы переобучения, дрейфа и качества данных не исчезают при переходе к LLM — они меняют форму.

## Ключевые термины урока

- **AI (Artificial Intelligence)** — широкая область создания систем, имитирующих интеллект.
- **ML (Machine Learning)** — подраздел AI: системы, обучающиеся на данных.
- **Deep Learning** — подраздел ML: многослойные нейросети с автоматическим извлечением признаков.
- **Generative AI** — подраздел Deep Learning: системы, создающие новый контент.
- **LLM (Large Language Model)** — большая языковая модель; Generative AI для текста.
