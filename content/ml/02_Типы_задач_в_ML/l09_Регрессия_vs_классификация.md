# 09. Регрессия vs классификация

## Зачем это нужно

Перед тем как выбрать модель и метрику, нужно определить тип задачи. Регрессия и классификация — два главных типа в supervised learning. От этого зависит, какой алгоритм брать (`LinearRegression` или `LogisticRegression`), какую метрику считать (MAE или Accuracy) и как трактовать результат.

## Простая интуиция

- **Регрессия** отвечает на вопрос «**сколько?**»: сколько стоит квартира, сколько минут займёт доставка, какая будет зарплата.
- **Классификация** отвечает на вопрос «**к какому классу?**»: спам или нет, купит или не купит, кот / собака / птица.

Если ответ — произвольное число, это регрессия. Если ответ — выбор из фиксированного списка категорий, это классификация.

## Что это такое

**Регрессия** — задача, где target `y` — непрерывное число. Примеры: `price`, `salary`, `temperature`, `delivery_time_minutes`.

**Классификация** — задача, где target `y` — метка класса из конечного набора. Примеры: `spam (0/1)`, `churn (0/1)`, `species (cat/dog/bird)`.

Важный нюанс: если target записан как `0/1`, это всё равно **классификация**, а не регрессия. Важен смысл, а не формат числа. Модель должна выбрать один из классов, а не вернуть значение «0.73».

## Как это работает

| | Регрессия | Классификация |
|---|---|---|
| Что предсказываем | число | класс |
| Пример target | `12400000` | `spam` / `not_spam` |
| Типичные модели | LinearRegression, RandomForestRegressor | LogisticRegression, RandomForestClassifier |
| Метрики | MAE, MSE, RMSE, R² | Accuracy, Precision, Recall, F1, ROC-AUC |

Это видно прямо в названиях моделей: `*Regressor` — для регрессии, `*Classifier` — для классификации.

## Практика

1. Определи тип задачи для каждого target: `температура воздуха`, `спам или не спам`, `стоимость авиабилета`, `вид транспорта (автобус / метро / такси)`, `опоздает ли поезд (0 или 1)`.
2. Почему задача «предсказать, уйдёт ли клиент (`churn = 0/1`)» — это классификация, хотя target записан числами?
3. Две задачи: (а) предсказать точный возраст человека по фото; (б) предсказать возрастную группу «до 18 / 18–40 / 40+». Какой тип у каждой и почему?
4. Коллега строит модель для предсказания цены квартиры и оценивает её метрикой Accuracy. В чём ошибка?
5. Мини-кейс: датасет с оценками товара от 1 до 5, нужно предсказать оценку. Какой вопрос надо задать заказчику, чтобы понять — это регрессия или классификация?

## Краткий итог

- Регрессия → число. Классификация → класс.
- `0/1` в target — это классификация, а не регрессия.
- Тип задачи определяет модель (`*Regressor` / `*Classifier`) и метрику.
- Метрики разные: MAE/RMSE для регрессии, Accuracy/F1 для классификации.
- Сначала пойми тип задачи — потом выбирай алгоритм.

## Ключевые термины урока

- **Регрессия** — тип задачи, где модель предсказывает непрерывное числовое значение.
- **Классификация** — тип задачи, где модель относит объект к одному из заданных классов.
- **Regressor** — класс модели для задачи регрессии (`*Regressor` в sklearn).
- **Classifier** — класс модели для задачи классификации (`*Classifier` в sklearn).
