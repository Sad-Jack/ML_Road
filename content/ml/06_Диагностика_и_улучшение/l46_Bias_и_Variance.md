# 45. Bias и Variance

## Зачем это нужно

Bias-variance tradeoff — фундаментальная концепция машинного обучения, объясняющая главное противоречие: чем сложнее модель, тем меньше ошибка на обучении, но больше риск переобучения. Понимание этого баланса помогает принимать осознанные решения о сложности модели и объёме данных.

## Простая интуиция

Представьте лучника. Высокий bias — он всегда промахивается в одну сторону (систематическая ошибка). Высокая variance — стрелы летят хаотично, каждый раз в разные места. Идеальный лучник — стрелы кучно попадают в цель: низкий bias и низкая variance.

## Что это такое

- **Bias (смещение)** — систематическая ошибка модели: насколько предсказания в среднем отклоняются от истинных значений. Высокий bias = underfitting
- **Variance (дисперсия)** — насколько сильно модель меняется при изменении обучающих данных. Высокая variance = overfitting
- **Tradeoff** — уменьшение bias обычно увеличивает variance и наоборот
- **Общая ошибка** = Bias² + Variance + Irreducible Noise
- **Irreducible noise** — шум в данных, который нельзя устранить никакой моделью

## Как это работает

```
# Bias-variance tradeoff (Decision Tree, max_depth):
#
# max_depth=1  → train=0.62, test=0.60   высокий bias,    низкая variance
# max_depth=4  → train=0.88, test=0.86   баланс           баланс
# max_depth=20 → train=0.99, test=0.72   низкий bias,     высокая variance
#
# Итог: сложность модели ↑ → bias ↓, variance ↑
# Irreducible noise остаётся при любой сложности
```

## Простой пример

Датасет из 500 медицинских снимков для предсказания наличия опухоли. Логистическая регрессия (высокий bias): RMSE одинаково плохо на train и test — модель слишком проста, не улавливает нелинейные паттерны. Дерево решений без ограничений (высокая variance): отлично на train, провал на test — запомнило шум в 500 примерах. Случайный лес (баланс): умеренно хорошо и там, и там. Irreducible noise — некачественные снимки с плохим разрешением, которые никакая модель не классифицирует верно.

## Краткий итог

- Bias и variance нельзя снижать одновременно: это фундаментальный trade-off, определяющий стратегию улучшения модели
- Высокий bias = underfitting, высокая variance = overfitting
- Общая ошибка = Bias² + Variance + Irreducible Noise
- Цель — найти баланс: достаточно сложная модель, но не переобученная

## Ключевые термины урока

- **Bias (смещение)** — систематическая ошибка: модель недостаточно сложна для данных.
- **Variance (дисперсия)** — нестабильность: модель слишком чувствительна к конкретным обучающим данным.
- **Bias-variance tradeoff** — обратная зависимость: снижая bias, увеличиваешь variance, и наоборот.
- **Irreducible noise** — неустранимая случайная ошибка, не связанная с моделью.
