# 21. Метрики регрессии: MAE, MSE, RMSE

## Зачем это нужно

Когда модель предсказывает числа, нужно измерить, насколько сильно она ошибается и в каких единицах. MAE, MSE, RMSE и R² — четыре основных метрики для регрессии. Каждая смотрит на ошибку немного по-своему — и выбор зависит от того, насколько критичны крупные промахи.

## Простая интуиция

Модель предсказала цену квартиры 10 млн, реальная — 12 млн. Ошибка — 2 млн. Метрики усредняют такие ошибки по всем объектам. Разница между ними — в том, как они «штрафуют» большие ошибки и в каких единицах выражаются.

## Что это такое

**MAE (Mean Absolute Error)** — средняя абсолютная ошибка: `mean(|y_true - y_pred|)`. Самая интуитивная: «в среднем модель ошибается на X рублей». Устойчива к выбросам.

**MSE (Mean Squared Error)** — среднеквадратичная ошибка: `mean((y_true - y_pred)²)`. Возводит ошибки в квадрат → большие ошибки штрафуются сильно сильнее. Единицы измерения «квадратные» (рубли²) — неинтуитивно.

**RMSE (Root MSE)** — `sqrt(MSE)`. Возвращает метрику в исходные единицы, но сохраняет чувствительность MSE к крупным ошибкам.

**R² (coefficient of determination)** — насколько модель лучше тривиального baseline «всегда среднее». R² = 1 — идеально, R² = 0 — как baseline, R² < 0 — хуже baseline.

## Как это работает

```python
import numpy as np
from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score

y_true = np.array([100, 120, 140])
y_pred = np.array([110, 100, 150])

mae  = mean_absolute_error(y_true, y_pred)        # 13.33
mse  = mean_squared_error(y_true, y_pred)         # 200.0
rmse = np.sqrt(mse)                                # 14.14
r2   = r2_score(y_true, y_pred)                    # ~0.25

# По шагам:
#   errors = [-10, 20, -10]
#   MAE  = (10 + 20 + 10) / 3 = 13.33
#   MSE  = (100 + 400 + 100) / 3 = 200
#   RMSE = sqrt(200)            = 14.14
```

Заметь: одна ошибка 20 в MSE даёт вклад 400 — в 4 раза тяжелее, чем ошибка 10. В MAE — всего в 2 раза.

## Простой пример

Прогноз зарплаты:
- MAE = 5000 ₽ → «модель в среднем ошибается на 5000 ₽» — легко объяснить заказчику.
- RMSE = 7000 ₽ → чуть выше MAE из-за нескольких крупных промахов.
- R² = 0.78 → модель объясняет 78% разброса в зарплатах.

Контекст важен: MAE = 5000 ₽ для прогноза чаевых — катастрофа; для прогноза цены квартиры — отлично.

## Краткий итог

- MAE: средняя абсолютная ошибка, в исходных единицах, устойчива к выбросам.
- MSE: квадратичная, сильно штрафует крупные ошибки, единицы — «в квадрате».
- RMSE: `sqrt(MSE)`, в исходных единицах, чувствительна к выбросам.
- R²: насколько модель лучше «всегда среднее»; 1 — идеал, 0 — как baseline.
- Без контекста число метрики ничего не значит — нужен масштаб target.

## Ключевые термины урока

- **MAE (Mean Absolute Error)** — средняя абсолютная ошибка в исходных единицах; устойчива к выбросам.
- **MSE (Mean Squared Error)** — среднеквадратичная ошибка; сильно штрафует крупные ошибки.
- **RMSE (Root MSE)** — корень из MSE; в исходных единицах, чувствительна к выбросам.
- **R² (коэффициент детерминации)** — доля дисперсии, объяснённой моделью; от 0 до 1.
