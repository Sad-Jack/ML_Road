# ML Road

`ML Road` — офлайн-курс в формате одного HTML-файла: интерактивная страница с уроками по Machine Learning и отдельным Python-треком, полнотекстовым поиском, навигацией по модулям и встроенной базой знаний (правая панель).

Пользователю нужен только **`ml_road.html`**. Всё остальное в репозитории — исходники уроков и инструменты сборки.

## Исходники

### Метаданные модуля (`_module.json`)

```json
{
  "id": "s1",
  "num": "01",
  "title": "База ML и данные",
  "color": "c1"
}
```

### Frontmatter урока

```yaml
---
id: l01
order: 1
num: "01"
title: "01. Что такое машинное обучение простыми словами"
short: "Что такое машинное обучение"
---
```

- `id` — уникальный идентификатор; становится именем `cdata`-блока в итоговом HTML
- `order` — целое число для сортировки внутри модуля
- `num` — отображаемый номер урока (строка в кавычках, ведущий ноль сохраняется)
- `title` — полный заголовок
- `short` — короткое название для сайдбара

### Устройство HTML

В итоговом `ml_road.html`:
- Структура курса хранится в JS-объектах `COURSE` и `PYTHON_COURSE`
- Содержимое уроков — в блоках `<script type="text/plain" id="cdata-lXX">`
- Статьи правой панели — в JS-массиве `RIGHT_PANEL_DATA`

## Сборка

Требования: **Python 3.9+**, стандартная библиотека (внешних зависимостей нет).

### Команды

**Полная пересборка** (template + content → ml_road.html):
```bash
python3 build.py --full
```

**Патч-режим** — обновить только конкретные уроки в существующем HTML (быстрее):
```bash
python3 build.py l15 l52
```

**Проверка** — убедиться что content/ совпадает с ml_road.html (без записи):
```bash
python3 build.py --check
```

## Рабочий процесс

### Редактирование урока

1. Откройте `.md` файл в `content/ml/<модуль>/`.
2. Внесите правки.
3. Пересоберите:
```bash
python3 build.py --full
```

### Добавление нового урока в конец модуля

1. Создайте файл `lXX_Название.md` в нужной папке модуля.
2. Добавьте frontmatter (id, order, num, title, short).
3. Пересоберите: `python3 build.py --full`.

### Вставка нового урока в середину курса

Используется `insert_lesson.py`. Скрипт атомарно сдвигает нумерацию всех уроков, начиная с позиции вставки, и обновляет внутренние перекрёстные ссылки.

Порядок:
1. Задайте позицию вставки в скрипте: `INSERT_AT = N` (все уроки с `order >= N` сдвинутся на +1).
2. Предпросмотр: `python3 insert_lesson.py --dry`
3. Примените: `python3 insert_lesson.py`
4. Создайте файл нового урока `lN_Название.md`.
5. Пересоберите: `python3 build.py --full`
